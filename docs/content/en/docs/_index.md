OpenSOHO is built to manage a small number of OpenWRT-based network devices. It is compatible with `openwisp-config` and `openwisp-monitoring`.

* From 2 to 20 devices (no hard limit)
* No multi-tenancy
* Simple to deploy

It is inspired by OpenWISP but targets networks that are too small to be maintained with OpenWISP.
